## Quick Steps:
1. Open Terminal at backend folder and execute: 
`npm run start`
2. Open Terminal at database folder and execute:
`sudo ./docker_create.sh`
`sudo ./docker_start.sh`
`./populate_db.sh` 
3. Open Terminal at frontend folder and execute: 
`npm run serve`
